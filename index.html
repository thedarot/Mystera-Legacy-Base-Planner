Mystera Legacy [Base Planner]

Coded by GreyHope / Updated by Darot
What is it?
Um planejador de bases para Mystera Legacy que calcula automaticamente o custo em materiais de suas construções.

How to use?
1. Select an item in the menu on the left
2. Click 'left' to put
3. Click 'right' to remove
4. Shift + Click to fill lines/retangles
5. Shift + Alt + Click to erase a large select area

How to save my project?
• Nomeie seu projeto no campo de texto
• Use "💾 Save" para salvar como .json editável
• Use "📂 Load" para carregar projeto anterior
• Use "Export" para gerar imagem ou texto

Enjoy!
